# Password Generator

A sleek, responsive, and interactive Password Generator built with **React** and styled using **Tailwind CSS**. This tool allows users to generate secure passwords instantly based on customizable criteria like length, numbers, and special characters.

## Features

*   **Dynamic Password Generation:** Automatically updates the password when criteria or length change.
*   **Custom Length Control:** Adjustable slider to select password length.
*   **Character Selection:** Toggle checkboxes to include or exclude numbers and special characters (`!@#$%^&*...`).
*   **One-Click Copy:** Seamlessly copy the generated password to your clipboard with instant visual feedback.
*   **Responsive UI:** Beautifully designed dark-themed interface built with Tailwind CSS that works flawlessly on all screen sizes.

## Tech Stack

*   **Frontend Library:** React (Hooks used: `useState`, `useCallback`, `useEffect`, `useRef`)
*   **Styling:** Tailwind CSS
*   **Build Tool:** Vite / Create React App
